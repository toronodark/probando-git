COMANDOS BASICOS DE GIT

git add.

agrega los archicos al stage

git commit -m "nombre del commit"

agrega los aarchivos al repositorio local

git push

sube los cambios al repositorio remoto

git pull

trae los al repositorio local lo q tenemos en el remoto

git branch

muestra las ramas que tenemos

git checkout

crear nueva rama (branchh)
